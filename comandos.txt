



Comando para clonagem de repositorio do GitHub
	git clone caminho para o repositorio

Comando para mostrar o status dos arquivos
	git status

Comando para mostrar o histórico de modificações
	git log
	git whatchanged

Configurar usuário e email
	git config user.name "nome"
	git config user.email "email"

Enviar arquivo para o repositorio remoto
	git push

Buscar alterações do repositóri remoto
	git pull
